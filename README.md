# SocialCal — Calendario Editoriale

## 🚀 Pubblicazione su Vercel (5 passi)

### PASSO 1 — Crea account GitHub
1. Vai su https://github.com e clicca **Sign up**
2. Inserisci email, password e username
3. Conferma l'email

### PASSO 2 — Carica il progetto su GitHub
1. Una volta loggato, clicca **"+"** in alto a destra → **New repository**
2. Nome: `socialcal` → clicca **Create repository**
3. Nella pagina del repo, clicca **"uploading an existing file"**
4. Trascina TUTTI i file e cartelle di questo ZIP (decomprimi prima)
5. Clicca **Commit changes**

### PASSO 3 — Crea account Vercel
1. Vai su https://vercel.com
2. Clicca **Sign Up** → scegli **Continue with GitHub**
3. Autorizza Vercel ad accedere a GitHub

### PASSO 4 — Pubblica l'app
1. Su Vercel clicca **"Add New Project"**
2. Seleziona il repository `socialcal`
3. Lascia tutto invariato → clicca **Deploy**
4. Aspetta ~2 minuti → l'app è online! 🎉

### PASSO 5 — Installa come app
**Su iPhone:**
1. Apri Safari → vai al link Vercel (es. socialcal-xxx.vercel.app)
2. Tocca l'icona Condividi (quadrato con freccia in su)
3. Scorri → **"Aggiungi a schermata Home"** → Aggiungi

**Su Android:**
1. Apri Chrome → vai al link
2. Tocca i 3 puntini in alto → **"Installa app"** oppure appare un banner in basso

**Su PC (Chrome/Edge):**
1. Apri il link
2. Clicca l'icona di installazione nella barra degli indirizzi (➕)
3. Clicca **Installa**
